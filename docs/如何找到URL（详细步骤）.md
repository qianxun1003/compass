# 如何找到URL（详细步骤）

## 🎯 目标

找到各大学官网的"外国人入試"或"留学生入試"页面URL，并填写到Excel表格中。

---

## 📝 一、在哪里填写？

### Excel表格（最简单）

**文件位置**：`crawled_data/大学URL填写表.xlsx`

**表格格式**：
| 大学名 | 主URL | 备注 |
|--------|-------|------|
| 東京大学 | | |
| 京都大学 | | |
| ... | | |

**填写步骤**：
1. 打开 `crawled_data/大学URL填写表.xlsx`
2. 找到对应的大学名
3. 在"主URL"列填写找到的网址
4. 如果需要，在"备注"列写说明
5. 保存文件

**就这么简单！**

## 🔍 二、什么是"主URL"？

### 简单说

**主URL = 各大学官网的"外国人入試"页面网址**

### 详细说

这个页面通常包含：
- ✅ 招生信息
- ✅ 出愿要求
- ✅ 考试信息
- ✅ 成绩要求
- ✅ 出愿时间
- ✅ 出愿材料
- ✅ 等等所有我们需要的信息

### 页面特征

**通常页面标题或内容会包含**：
- "外国人入試"
- "留学生入試"
- "International Students"
- "外国人特別選抜"
- "私費外国人留学生入試"

## 📖 三、具体操作步骤（以東京大学为例）

### 步骤1：打开大学官网

**操作**：
1. 打开浏览器（Chrome、Safari、Firefox等都可以）
2. 在地址栏输入：`www.u-tokyo.ac.jp`
3. 按回车访问

**结果**：看到東京大学的官网首页

### 步骤2：找到"入試"菜单

**操作**：
1. 查看网站顶部导航栏
2. 找到"入試"或"Admissions"相关菜单
3. 鼠标悬停或点击

**東京大学示例**：
- 顶部菜单有："入試・進学"
- 点击进入

**如果找不到**：
- 可以查看网站底部的"サイトマップ"（网站地图）
- 或者在网站搜索框中搜索"外国人入試"

### 步骤3：找到"外国人入試"页面

**操作**：
1. 在"入試"菜单下，查找"外国人入試"或"International Students"
2. 点击进入

**東京大学示例**：
- "入試・進学" → "学部入試" → "外国人入試"
- 点击"外国人入試"

**结果**：进入外国人入試页面

### 步骤4：复制URL

**操作**：
1. 确认这是正确的页面（应该能看到招生信息、出愿要求等）
2. 查看浏览器地址栏（页面顶部）
3. 选中整个网址
4. 复制（Ctrl+C 或 Cmd+C）

**東京大学示例**：
- 地址栏显示：`https://www.u-tokyo.ac.jp/admission/undergraduate/foreign.html`
- 复制这个网址

### 步骤5：填写到Excel

**操作**：
1. 打开 `crawled_data/大学URL填写表.xlsx`
2. 找到"東京大学"这一行
3. 在"主URL"列粘贴刚才复制的网址
4. 保存文件

**完成！**

## ❓ 五、常见问题解答

### Q1: URL很长怎么办？需要全部复制吗？

**A**: 
- ✅ **是的，需要复制完整的URL**
- 例如：`https://www.u-tokyo.ac.jp/admission/undergraduate/foreign.html?lang=ja&tab=requirements`
- 这样的长URL也可以，爬虫会自动处理

### Q2: 如果找不到"外国人入試"页面怎么办？

**A**: 
可以尝试：
- 搜索"International Students"
- 查找"私費外国人留学生"
- 或者找到"入試要項"（招生简章）页面也可以
- 如果实在找不到，可以先留空，后续一起处理

### Q3: 需要填写多个URL吗？

**A**: 
- ❌ **不需要！**
- ✅ **只需要一个主URL**
- 爬虫会自动从主页面查找其他相关信息
- 如果某些信息在特定页面，可以后续补充

### Q4: 如何确认URL是否正确？

**A**: 
打开URL后，应该能看到：
- ✅ 招生信息
- ✅ 出愿要求
- ✅ 考试信息
- ✅ 成绩要求
- ✅ 出愿时间
- ✅ 等等

如果页面主要是这些内容，就是正确的！

### Q5: 如果URL打不开怎么办？

**A**: 
- 检查网络连接
- 确认URL是否正确复制
- 如果确实打不开，可以在"备注"列标注"无法访问"
- 后续可以一起处理

### Q6: 可以分批填写吗？

**A**: 
- ✅ **完全可以！**
- 建议先填写前20所大学（记录数最多的）
- 然后每天填写10-20所
- 不需要一次性完成所有
