# 飞书表格特殊成绩要求填写指南

## 📋 一、需要添加的列

在飞书多维表格的**学校总览表**中，添加一个新列：

**列名**：`特殊成绩要求`

**列类型**：文本（单行文本）

---

## 📝 二、填写格式

### 2.1 基本格式

格式：`科目名:最低分数`

**科目名标准**（必须使用以下名称）：
- `日语` - 日语科目
- `数学1` - 数学コース1
- `数学2` - 数学コース2
- `综合` - 综合科目
- `物理` - 物理
- `化学` - 化学
- `生物` - 生物
- `托福` - TOEFL

### 2.2 单个要求示例

| 大学 | 学部 | 特殊成绩要求 |
|------|------|-------------|
| 東京大学 | 文科一類 | `数学1:200` |
| 早稲田大学 | 人間科学学部 | `数学1:150` |
| 某大学 | 某学部 | `日语:300` |

**说明**：
- `数学1:200` 表示数学1必须满分（200分）
- `数学1:150` 表示数学1必须≥150分
- `日语:300` 表示日语必须≥300分

### 2.3 多个要求示例

如果有多个科目的要求，用**逗号**分隔：

| 大学 | 学部 | 特殊成绩要求 |
|------|------|-------------|
| 某大学 | 某学部 | `数学1:150,日语:300` |
| 某大学 | 某学部 | `数学1:200,托福:90` |
| 某大学 | 某学部 | `数学1:150,日语:300,托福:90` |

**说明**：
- 多个要求用逗号分隔
- 逗号后可以有空格，也可以没有
- 例如：`数学1:150,日语:300` 或 `数学1:150, 日语:300` 都可以

### 2.4 满分对照表

如果需要填写"满分"要求，使用以下分数：

| 科目 | 满分 | 填写示例 |
|------|------|---------|
| 数学1 | 200 | `数学1:200` |
| 数学2 | 200 | `数学2:200` |
| 综合 | 200 | `综合:200` |
| 日语 | 450 | `日语:450` |
| 物理 | 100 | `物理:100` |
| 化学 | 100 | `化学:100` |
| 生物 | 100 | `生物:100` |
| 托福 | 120 | `托福:120` |

## ✅ 三、填写规则

1. **只填写有特殊要求的大学/学部**
   - 如果某个大学/学部没有特殊成绩要求，该列**留空**即可
   - 不需要填写"无"或"无要求"

2. **科目名必须准确**
   - 必须使用标准科目名（见2.1节）
   - 不能使用其他名称，如"数学"、"Math1"、"日语科目"等

3. **格式必须正确**
   - 格式：`科目名:分数`
   - 冒号必须是英文冒号 `:`
   - 分数必须是数字（可以是小数，如 `150.5`）

4. **多个要求用逗号分隔**
   - 格式：`科目1:分数1,科目2:分数2`
   - 逗号后可以有空格，也可以没有

---

## ❌ 四、错误示例

以下填写方式是**错误的**，请避免：

| ❌ 错误示例 | ✅ 正确写法 | 错误原因 |
|------------|------------|---------|
| `数学1必须150分` | `数学1:150` | 格式不正确，应使用冒号 |
| `数学1≥150` | `数学1:150` | 不能使用≥符号 |
| `Math1:150` | `数学1:150` | 科目名错误，应使用中文 |
| `数学:150` | `数学1:150` 或 `数学2:150` | 科目名不明确 |
| `数学1:150分` | `数学1:150` | 分数后不能加单位 |
| `数学1必须满分` | `数学1:200` | 应填写具体分数 |

## 📋 五、填写步骤

1. **打开飞书多维表格**
   - 找到"学校总览表"

2. **添加新列**
   - 点击表格右上角的"+"按钮
   - 选择"添加列"
   - 列名填写：`特殊成绩要求`
   - 列类型选择：文本（单行文本）

3. **填写数据**
   - 对于有特殊要求的大学/学部，按照格式填写
   - 对于没有特殊要求的大学/学部，留空即可

4. **检查格式**
   - 确认科目名使用标准名称
   - 确认格式为 `科目名:分数`
   - 确认多个要求用逗号分隔

## 💡 六、常见问题

### Q1: 如果某个大学要求"数学1必须满分"，应该怎么写？
**A**: 填写 `数学1:200`（数学1的满分是200分）

### Q2: 如果某个大学要求"数学1≥150分"，应该怎么写？
**A**: 填写 `数学1:150`

### Q3: 如果某个大学有多个要求，比如"数学1≥150且日语≥300"，应该怎么写？
**A**: 填写 `数学1:150,日语:300`（用逗号分隔）

### Q4: 如果某个大学没有特殊要求，需要填写什么？
**A**: **留空即可**，不需要填写任何内容

### Q5: 科目名可以写错吗？
**A**: **不可以**。必须使用标准科目名（日语、数学1、数学2、综合、物理、化学、生物、托福），否则系统无法识别。

### Q6: 分数可以是小数吗？
**A**: 可以，例如 `数学1:150.5`，但通常使用整数。
